## Proyecto destacado

### Análisis completo de 3 millones de viajes de taxis en NYC (Enero 2023)  
Dataset oficial NYC TLC – Yellow Taxi Trips

**Objetivo**: Entender patrones de demanda, comportamiento de propinas, impacto del tráfico y composición de costos.

**Tecnologías usadas**:
- Python · Pandas · Matplotlib · Seaborn
- Google Colab · GitHub

**Insights clave de negocio**:
| Pregunta                          | Respuesta con datos                              | Impacto sugerido                     |
|-----------------------------------|--------------------------------------------------|--------------------------------------|
| ¿A qué hora hay más viajes?       | Pico máximo a las 18:00 hs (180k viajes)         | Más conductores 16–20 hs             |
| ¿Quién da más propina?            | Tarjeta: ~21% → Efectivo: 0%                     | Promover pago con tarjeta           |
| ¿Dónde está el peor tráfico?      | Zona 86 → viajes de 55 min promedio             | Precios dinámicos o reruteo          |
| ¿Cuánto duran los viajes en hora pico? | +70% vs madrugada                            | Informar espera real al pasajero    |
